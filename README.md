3. 디테일 페이지에서 사용자와 상호작용을 위해 설명 더보기/접기 기능 좋아요 표시, 댓글 입력기능 구현
4. 마이 페이지에서 이름, 자기소개, MBIT, 취미 등 편집기능, 이미지를 눌렀을 때 이미지를 선택해서 바꾸는 기능 구현
